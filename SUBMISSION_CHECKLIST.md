- Open the app locally and confirm it starts:

```powershell
streamlit run app.py
```

- Check that the main app sections are working:
  - Road to Abtal
  - Parent Story Studio
  - My Story Maker / Dream Career Adventures
  - Activity Companion
  - Summer Camp Extension
  - Parent Space

- Make sure the README clearly explains:
  - What the project does
  - Why it matters
  - Main features
  - How to install and run it
  - What environment variables are needed

- Confirm `.env.example` has placeholders only, not the real Fanar API key.
